Setup

Up and running
in 30 minutes.

We handle the setup. You just watch the dashboard light up.

01
Install the Server

Run the one-click setup on your main office PC — the one that stays on. Node.js powers the dashboard and auto-starts on every Windows reboot. Works on Windows 7, 8, 10 and 11.

02
Deploy Employee Agents

Generate a silent installer for each employee in seconds. Copy the folder to their PC, run it once. The agent installs itself invisibly and starts reporting immediately.

03
Open Your Dashboard

Open any browser on your network and go to your server's IP address. Log in and instantly see every employee's live status, attendance, apps, and timesheets — all in one view.
